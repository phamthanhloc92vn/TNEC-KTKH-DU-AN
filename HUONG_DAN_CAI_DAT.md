# CONTRACT AI - Trungnam E&C
## Hướng dẫn cài đặt

### Yêu cầu
- **Node.js** phiên bản 18 trở lên: https://nodejs.org/en/download

### Cài đặt
1. Giải nén file ZIP vào thư mục bất kỳ (ví dụ: `C:\ContractAI`)
2. Double-click file **`START.bat`** để khởi động

### Sử dụng
- Mở trình duyệt vào: **http://localhost:3000**
- Lần đầu chạy: vào **Settings** (⚙️) nhập OpenAI API Key
- Kéo thả file PDF hợp đồng vào ứng dụng

### Dừng ứng dụng
- Đóng cửa sổ dòng lệnh (CMD) là ứng dụng tắt

### Lưu ý
- API Key được lưu trong trình duyệt (localStorage), mỗi máy cần nhập lại một lần
- Cần kết nối Internet để gọi OpenAI API và đồng bộ Google Sheets
